## VS Code
VS Code is a program used to write and edit code. It has extensions that help in coding and make work easier.It is used to create and openHTML files.

## Git
Git is a tool that helps track changes in code. It allows one to  commit and push their  work to GitHub

## Node.js
Node.js allows JavaScript to run outside the browser. It also comes with npm which helps install packages.

## Terminal
The terminal is where commands are typed to control the computer and projects. I used it to run Git commands and check versions of installed tools. It is important because many developer tasks are done through it.
