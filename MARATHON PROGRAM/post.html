What I Learned in Week 1 at Mctaba Labs

Published on 30 April 2026

My Learning Experience

In Week 1 of learning web development, I was able to understand and set up different apps used by developers and their functions, such as VS Code and Git.

I also learned about semantic HTML, which helps browsers and developers understand what each part of a webpage means. Using semantic tags makes code more organized and readable.

Another important lesson was how to create links between pages. I now understand how websites are connected using navigation menus, making it possible to move from one page to another easily.
