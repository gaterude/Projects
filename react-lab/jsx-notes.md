JSX is an extention of Javascript with a mix of HTML used in resact while building apps 
JSX is not real HTML. It is changed into JavaScript code behind the scenes using React.createElement().
For example, <h1>Hello</h1> becomes React.createElement("h1", null, "Hello").
React uses JSX to show things on the screen, and it updates the page automatically when data (state) changes. This makes building user interfaces easier
